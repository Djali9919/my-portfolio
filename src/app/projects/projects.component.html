Book Store

Unlock the mystery one book at a time

An online portal to discover and buy books. Here you can filter out books based on Author and Genre and also sort them in the order of their publishing dates. The UI is powered with Angular you provide an interactive, user-friendly single page web application. The backend utilizes the microservice architecture using the Spring Boot. For data storage Azure SQL data base is used.

Weather Forecast Mobile App

A Freelance Project

Developed this app for Android devices as a Freelancer. It was different from other weather forecast apps because in my app I used Analog meters to display metrics like Temperature and Atmospheric pressure of any place worldwide. I used Google Weather API to get the accurate data.
